# Spam Detection using RNN

This project implements a spam detection model using Recurrent Neural Networks (RNN) with LSTM layers. The model is trained on a dataset of SMS messages to classify them as spam or ham (not spam).

## Features

- Data preprocessing and cleaning
- LSTM-based RNN model for text classification
- Model evaluation with accuracy, precision, recall, and F1-score
- Visualization of training history, model comparison, and confusion matrices
- Testing on new example sentences and messages

## Requirements

- Python 3.x
- TensorFlow
- Keras
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Usage

Run the script `Spam Detection باستخدام RNN.py` to train the model and evaluate it on the test set.
